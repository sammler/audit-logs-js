# s5r-audit-logs

> Node.js library to send audit-logs to the s5r/auth-service.

---

## Installation

```
$ npm install s5r-audit-logs
```

## Usage

```
const AuditLogger = require('s5r-audit-logs');

const natsOps = {};

let auditLogger = new AuditLogger(natsOpts);
let subject = 'foo';
let msg = {};

auditLogger.log(subject, msg);
```
